## Updating the Website
Check if you have [Node.js and npm](https://nodejs.org/en/download/) installed by entering:
```
node --version
npm --version
```
### Locally
1. cd to the `frontend` project folder and run:
```
npm start
```
2. The website will launch in your browser
4. Once done updating, push changes to repo
### GitHub Pages
1. Make changes
2. cd to the `frontend` project folder and run:
```
npm run deploy
```
3. Wait until the website is deployed again
